Joining the Headstarter Fellowship

This summer, I got accepted into the Headstarter AI SWE Fellowship, a New York-based program that was a game-changer for me, even though I’m based in Pakistan. The time difference meant late-night meetings and conferences, but it was worth it. I connected with amazing teammates from other universities, learned a ton about AI and web development, and built some cool projects. One of my favorites was the AI Roadmap Generator, a Streamlit app we created as a team.

Building the AI Roadmap Generator

The AI Roadmap Generator (try it here) is a web app that uses Google’s Gemma2-9b open-source model to generate personalized AI learning roadmaps. We integrated Firebase for user authentication and data storage, all built with Streamlit for a quick and clean frontend. Streamlit made it so easy to turn my Python skills into a functional web app without diving too deep into complex web dev.

Here’s a snippet of how we set up the basic Streamlit interface:


import streamlit as st

st.title("AI Roadmap Generator")
st.write("Generate a personalized AI learning roadmap with Gemma2-9b!")
user_input = st.text_input("Enter your AI learning goals:")
if st.button("Generate Roadmap"):
    # Placeholder for Gemma2-9b model integration
    st.write("Your roadmap is being generated...")

This code sets up a simple interface where users input their goals, and the app (with backend logic) generates a roadmap. The real magic happens with the Gemma2-9b model and Firebase, but Streamlit’s simplicity let us focus on functionality over frontend fuss.

Last-Minute Bugs and Fixes

Building the app wasn’t all smooth sailing. We hit a nasty bug with Firebase authentication where sessions weren’t persisting across page refreshes. After hours of debugging, we found a misconfigured Firebase SDK call. Another headache was the Gemma2-9b model’s response time, it was sluggish on initial loads. We optimized the backend to cache common queries, which sped things up. These late-night fixes taught me how critical testing and iteration are in real-world projects.

Teamwork and Deployment

I worked with an awesome team, and I have to shout out Awwab, who handled the deployment like a pro. We met through the fellowship and collaborated. Awwab set up the Streamlit app on their hosting platform, ensuring it was live at ai-roadmap-generator.streamlit.app. The teamwork aspect of the fellowship pairing with folks from different unis—was a highlight and showed me how diverse skill sets come together in software engineering.
