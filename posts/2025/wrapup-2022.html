2022 Recap: The Year I Chased Logic and Found Myself

How It Started

2022 was a wild year. I started learning the basics of computer science, right after winning a competition in my college’s CS department. The crazy part? I was still a chemistry pre-engineering student then. I chose CS over AI because it felt like the better starting point (check my first post). I'm just a guy trying out code, barely knowing what I was doing. By December, running this tech blog, and working with Python, Java, and some cool new stuff. That’s my 2022 messy, tough, but full of moments that shaped me.

I started with CodeWithHarry and Apna College.

Python and Java Days

In May, I was grinding through Python and Java courses, and Python became my favorite, thanks to Zia Sir’s awesome classes (I wrote about it in my Python post). Python was easy, less typing, more doing. I built a small script to track my study hours for a college assignment, and it felt like magic. Java was okay, but way too much work for simple stuff. By mid-year, I’d finished both courses, which was a big deal for someone who started from scratch.

Below is my first assignment code given by my Sir :> Still Remember this!!

import pandas as pd

# Simple script to track study hours (used for a python assignment)
data = {'Date': ['2022-05-01', '2022-05-02', '2022-05-03'],
        'Hours': [2, 3, 1.5]}
df = pd.DataFrame(data)
print(df)
print(f"Total hours studied: {df['Hours'].sum()}")

LeetCode Chaos

By September, I thought I’d level up with Data Structures and Algorithms on LeetCode. Wrong move. I shared the struggle in my DSA post. Arrays and loops drove me nuts, and college exams made it worse. I’d spend hours on one problem, get “wrong answer,” and want to give up. Burnout hit, but Zia Sir’s tips and small wins kept me going. I’m not a DSA pro yet, but I’m still in the fight.

ChatGPT Saves the Day

Then, November 2022 brought ChatGPT. Man, it blew my mind and I could say that this thing was a lifesaver. I’d ask it why my Python code crashed or how to solve a LeetCode problem, and it explained it like a friend. It even helped with college assignments figuring out logic for a Java project or breaking down loops for me. It made coding less scary and got me thinking about what AI can do. As a blog writer, I know I’ll be writing about this a lot.

What I Learned

2022 was rough but worth it. I learned Python and Java, survived LeetCode, and found ChatGPT. College stress taught me I can’t do everything at once. I’m ending the year with two languages, a blog I love, and big questions about AI’s future. The biggest takeaway? Keep going, even when it feels impossible.

For 2023, I’m taking it slow. No more late-night panic coding. I’ll keep blogging, coding, and exploring AI, one step at a time.

Every error is a lesson. Keep coding, keep learning, keep moving forward.
