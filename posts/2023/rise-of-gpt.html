Messing with NLP in Python

ChatGPT got me curious about how it understands my questions, so I started playing with Natural Language Processing (NLP). I used Python with NLTK and Scikit-learn to build a basic text classifier that sorts movie reviews into positive or negative. It’s not fancy, but seeing it work felt like I was cracking some secret code. Python’s libraries make this stuff doable, even for a guy like me who’s still dodging calculus.

Here’s a bit of the code I messed with:


import nltk
from nltk.corpus import movie_reviews
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB

# Basic NLP text classifier
nltk.download('movie_reviews')
documents = [(list(movie_reviews.words(fileid)), category)
             for category in movie_reviews.categories()
             for fileid in movie_reviews.fileids(category)]
texts = [' '.join(doc) for doc, _ in documents]
labels = [cat for _, cat in documents]
vectorizer = TfidfVectorizer(max_features=1000)
X = vectorizer.fit_transform(texts)
clf = MultinomialNB().fit(X, labels)
print("Classifier trained!")
